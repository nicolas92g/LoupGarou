# LoupGarou
interface graphique qui permet de jouer au loup garou et qui fonctionnent avec opengl !

librairie utilisées : glfw, glm, glad, std-image, freetype

-enum roles et une fonction qui repartis les roles en fonction du nombre de joueurs		(thibault)

-une fonction par role pour designer ce que le role fait pendant la nuit				(thibault et david)

-une fonction qui appelle les fonctions de chaque role									(thibault et david)

-une fonction pour le vote																(nicolas)

-une fonction qui indique que le jeu est terminé										(nicolas)

-une fonction qui affiche le gagnant													(nicolas)
